Bootstrap for designers!

Framework for building simple and beautiful home pages. It's an amazing tool to fast track your designs and facilitate web development.

Strong foundation

Open source toolkit for developing with HTML, CSS, and Javascript. It provides a strong foundation for developers and designers to quickly prototype ideas and build responsive websites.

View documentation »

Fits any process

Flexible framework that fits any process. Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to Sass variables and mixins, and dozens of predefined classes.

View themes »

Plug and Play

It's easy to just start using the tool. There's an extensive components library that includes buttons and lists to carousels and modals. Components are easy to integrate and require no knowledge of JavaScript to use.

View live examples »

Colors

Bootstrap has an accessible default color palette. You can use the colors anyway you like but the framework has default names for each of them. You can use a primary color for interactive elements, such as links and buttons. Secondary, success, warning, and alert colors can also be used to give more context to UI elements and actions.

Cards

A card is a flexible container for content. There are options for headers and footers, a wide variety of content, contextual background colors, and powerful display options. Cards are built with as little markup and styles as possible, but still manage to deliver a ton of control and customization.

  • Code

Boost your design process

Using Bootstrap can actually save a lot of time. You won't be writing a lot of code as you can just use the predefined design templates and classes to design everything the way you want. That’s why it’s simple to use. If you have the basic working knowledge of HTML and CSS, you can start a consistent development with Bootstrap.
